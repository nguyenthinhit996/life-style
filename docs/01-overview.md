# Project Overview

## What Is This?

**Life-Style** is a personal brand + blog website built by Peter.
It is a public-facing platform combined with a private admin dashboard, covering two core content pillars:

- **Information Technology** — tutorials, tools, coding tips, software, and tech news
- **English** — grammar guides, vocabulary, writing tips, and communication skills

The goal is to build a fully custom, self-managed website — no third-party CMS, no WordPress — everything is owned and built from scratch.

---

## Goals

| Goal | Description |
|------|-------------|
| Personal Brand | Present Peter's identity, skills, and story to the world |
| Blog Platform | Publish articles on IT and English topics for the general public |
| Admin Dashboard | Privately write, edit, and manage blog posts from a custom dashboard |
| Modern Design | Bold, colorful, and visually impressive UI |
| Fast Launch | Ship basic version quickly, then iterate |

---

## What Makes It Different?

- Fully custom admin dashboard — no third-party CMS
- Two content worlds (IT + English) under one personal brand
- Bold & colorful design — not a typical minimalist blog
- Built with the latest web stack (Next.js App Router, Tailwind CSS, TypeScript)

---

## Core User Types

| User | Description |
|------|-------------|
| **Visitor (Public)** | Reads blog posts, explores the personal brand page |
| **Admin (Peter)** | Logs in, writes posts, manages content from the dashboard |

---

## Success Definition

- The website is live at a custom domain
- Peter can log in and publish new blog posts without touching code
- Visitors can discover, read, and share content easily
- The site looks professional and represents Peter's brand well
