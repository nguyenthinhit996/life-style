Hello World: Your First Java Program

☕ Java Fundamentals IT 🕐 6 min read 📅 Jan 12, 2026

Congratulations on getting Java installed! Now it is time to write your very first program. In this lesson we will write the classic Hello, World! program, understand what every line means, and run it successfully.

Create Your First Java File

In Java, every program lives inside a class. The file name must match the class name exactly. Create a new file called HelloWorld.java and type the following:

Java 
// HelloWorld.java
public class HelloWorld {

    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }

}

Breaking It Down

Let us look at each part of this program so nothing feels like magic:

Explanation 
public class HelloWorld   → declares a class named HelloWorld
main(String[] args)       → the entry point Java looks for to start
System.out.println(...)   → prints text to the console
"Hello, World!"           → a String literal (text value)

Run It

In IntelliJ IDEA, click the green ▶ button next to the main method. You should see the following output in the terminal at the bottom:

Output
Hello, World!

That's it — you just ran your first Java program! In the next lesson we will explore how Java actually compiles and executes this code under the hood using the JVM.
